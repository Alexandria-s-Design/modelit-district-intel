## Funding Eligibility for EdTech Purchases

Calexico Unified's demographics make it eligible for multiple funding streams that can cover EdTech:

| Funding Source | EdTech Eligibility | Calexico Qualification |
|---------------|-------------------|----------------------|
| Title IV-A (SSAE) | Yes — explicit EdTech and STEM use | Qualifies as high-poverty Title I district |
| LCFF Supplemental (EL/SED/Foster) | Yes — for services to unduplicated pupils | ~88% SED + ~60% EL = massive concentration grant |
| LCFF Concentration | Yes | Districts with >55% unduplicated pupils get concentration funds |
| E-rate | Yes — for connectivity (infrastructure) | Likely already used for network |
| Governor's STEM initiatives | Yes — if active CA grants apply | Check current CDE grant opportunities |
| Measure Q Bond (capital) | Technology if classified as capital improvement | New STEM buildings = natural tech integration ask |
